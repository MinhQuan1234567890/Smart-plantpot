# Smart-plantpot

1. Overview
-A plantpot that moniter the soil's moisture and automatically water itself if it is dry and stop when have sufficient moisture
-Why and goal: To help save time while also having an accurate reading of the soil's moisture
2.Hardware and circuit
-One water pump
-One moisture sensor
-One relay
-One WeMos D1 R1
-One breadboard
-One battery pack
-A lots of wires
